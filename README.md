# Vladimir AI — лендинг AI-агентства

Одностраничный лендинг агентства по AI-автоматизации.

## Состав
- `index.html` — разметка лендинга
- `styles.css` — стили (палитра взята с обложек YouTube: кремовый фон, графит, кораллово-оранжевый акцент)

## Запуск
Открыть `index.html` в браузере. Сборка не нужна — чистый HTML/CSS, шрифты подключены через Google Fonts.
